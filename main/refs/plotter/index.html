    arkas.plotter.BasePlotter.equal abstractmethod ¶
    equal(other: Any, equal_nan: bool = False) -> bool

    Indicate if two plotters are equal or not.

    Parameters:

    Name Type Description Default
    other Any

    The other plotter to compare.

    		 required
    equal_nan bool

    Whether to compare NaN's as equal. If True, NaN's in both objects will be considered equal.

    		 False

    Returns:

    Type Description
    bool

    True if the two plotters are equal, otherwise False.

    Example usage:

    >>> from arkas.plotter import Plotter
>>> plotter1 = Plotter()
>>> plotter2 = Plotter()
>>> plotter3 = Plotter({"fig": None})
>>> plotter1.equal(plotter2)
True
>>> plotter1.equal(plotter3)
False
